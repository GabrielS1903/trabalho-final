Sobre o Projeto

Este é um projeto de exemplo para demonstrar um sistema básico de cadastro de filmes e categorias.

Utiliza Spring Boot no backend e HTML, CSS e JavaScript no frontend.

Funcionalidades

  • Cadastrar novas categorias de filmes.
  • Cadastrar novos filmes associados a uma categoria existente.
